RESISTORS

PREFERRED VALUES
It would not be practical to make resistors of every value, so a system of preferred values is used. This system sets a fixed number of values in each decade of resistance, each value being an increase over the previous value by an approximately constant percentage. For example, if the values in the first decade of a preferred value series are 10, 22, and 47 ohms then the values in the next decade would be 100, 220, and 470 ohms, for the decade after 1K, 2k2, and 4k7 ohms and so on.
